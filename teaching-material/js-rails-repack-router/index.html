JavaScript / Rails - Repack React Router

This tutorial will go over how to make an app using Rails as the JSON API backend and React / React Router as the Client Frontend. We will be using the Repack Gem (https://github.com/cottonwoodcoding/repack) to help us get React and Router configured in the industry standard way so we can treat JavaScript as a first class citizen in a Rails Ecosystem.

This is the way to install Repack for basic React use only

This is the way to install Repack for React use with React Router

Rails Routes - config/routes.rb

Rails.application.routes.draw do
    root 'home#landing'
    namespace :api do
        get 'beers', to: 'beer#index'
        get 'billboard', to: 'beer#billboard'
    end
    # FOR REACT ROUTER - NO ROUTES BELOW THIS!!!
    get '*unmatched_route', to: 'home#landing'
end
We are using an api namespace so our React Router routes and the Rails routes donâ€™t get mixed up

Billboard Component - client/components/Billboard.js

import React, { Component } from 'react';
import Beers from './Beers';
class Billboard extends Component {
    constructor(props) {
        super(props);
        this.state = { beers: [] };
    }
    componentDidMount() {
        $.ajax({
            url: '/api/billboard',
            type: 'GET',
            dataType: 'JSON'
        }).done( beers => {
            this.setState({ beers });
        }).fail( data => {
            // TODO: show the user some sort of error
            console.log(data);
        });
    }
    render() {
        return(
            <div>
              <h2>Top 100 Beers</h2>
              <hr />
              <div className='row'>
                <Beers beers={this.state.beers} />
              </div>
            </div>
        );
    }
}
export default Billboard;

Beers Component - client/components/Beers.js

import React, { Component } from 'react';
import Beer from './Beer';
class Beers extends Component {
    constructor(props) {
        super(props);
    }
    displayBeers() {
        let beers = this.props.beers;
        if(beers.length) {
            return beers.map( beer => {
                return(<Beer key={beer.id} beer={beer} />);
            });
        } else {
            return(<h3>No Billboard Beers</h3>);
        }
    }
    render() {
        return(
            <div>
              { this.displayBeers() }
            </div>
        );
    }
}
export default Beers;

Running The App

You need the Rails server running and the WebPack Dev Server running

Running The Rails Server

bundle exec rails s

Running The Webpack Dev Server

There is 2 different ways of runing the Webpack Dev Server

Yarn Installed - https://github.com/yarnpkg/yarn

yarn run dev_server

Yarn NOT Installed - https://www.npmjs.com/

npm run dev_server
